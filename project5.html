Dice Game


This HTML document sets up a simple web page called "Dicee," designed to simulate a dice-rolling game. The page includes a header with the text "Refresh Me," indicating that the dice can be rolled by refreshing the page. There are two sections for players, labeled "Player 1" and "Player 2," each with an image of a dice face, initially displaying six dots. The dice images are sourced from a local "images" folder. The page links to an external CSS file, "styles.css," for styling and uses Google Fonts for custom fonts. It also includes a reference to a JavaScript file, "index.js," which is likely used to implement the functionality for randomly displaying different dice images when the page is refreshed.
